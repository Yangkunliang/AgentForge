# TASK-004 产品需求：Skill 管理、Dashboard、费用统计与数据导出

## 1. 背景

TASK-001/TASK-002/TASK-003 已完成基础认证、任务管理、Agent 管理和 Harness 核心。现在需要补充：
- Skill 插件系统的完整 API
- Dashboard 数据展示 API
- 费用统计与成本追踪 API
- 训练数据导出 API

## 2. 功能需求

### 2.1 Skill 管理 API

| 功能 | 描述 |
|------|------|
| 安装 Skill | 从 Git/PyPI 安装 Skill，支持异步安装和进度查询 |
| 查询安装进度 | 实时查询 Skill 安装状态和日志 |
| 列出已安装 Skill | 获取所有已安装 Skill 的列表 |
| 卸载 Skill | 从系统中移除指定 Skill |

### 2.2 Dashboard API

| 功能 | 描述 |
|------|------|
| 全局概览 | 任务统计、Agent 状态、Skill 数量、费用趋势 |
| 最近任务 | 展示最近完成的任务列表 |

### 2.3 费用统计 API

| 功能 | 描述 |
|------|------|
| 每日费用 | 查询指定日期的总费用和各模型费用分布 |
| 任务统计 | 当日任务数和平均每任务费用 |

### 2.4 数据导出 API

| 功能 | 描述 |
|------|------|
| 发起导出 | 创建训练数据导出任务 |
| 查询导出状态 | 追踪导出进度 |
| 下载导出文件 | 获取导出的 JSONL 文件 |

## 3. 非功能需求

- API 响应时间 < 200ms（Dashboard 聚合查询 < 500ms）
- 导出任务支持断点续传
- 数据脱敏符合 GDPR 标准
