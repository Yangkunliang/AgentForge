# TASK-005 产品需求：前端工作台 & UI/UX 体验

## 1. 背景

TASK-001 至 TASK-004 完成了后端 API、Harness 核心、Skill 系统和 Webhook 回调。现在需要构建前端工作台，让用户能够：
- 登录注册
- 创建和管理任务
- 实时查看任务执行进度（SSE）
- 管理 Agent 和 Skill
- 查看 Dashboard 统计
- 导出训练数据

## 2. 功能需求

### 2.1 认证模块

| 功能 | 描述 |
|------|------|
| 用户注册 | 表单验证、密码强度 |
| 用户登录 | Token 管理、自动刷新 |
| 退出登录 | 清除本地状态 |
| 权限控制 | 路由守卫、按钮级权限 |

### 2.2 任务管理

| 功能 | 描述 |
|------|------|
| 创建任务 | 输入描述、选择优先级 |
| 任务列表 | 分页、筛选、状态标签 |
| 任务详情 | 子任务、Agent 竞标、实时日志 |
| 任务反馈 | 评分、评论 |

### 2.3 实时事件

| 功能 | 描述 |
|------|------|
| SSE 连接 | fetch + ReadableStream |
| 事件展示 | 任务拆解、Agent 竞标、Skill 调用 |
| 自动重连 | 指数退避 |

### 2.4 Agent/Skill 管理

| 功能 | 描述 |
|------|------|
| Agent 列表 | 状态、capability 筛选 |
| Agent CRUD | 需 admin 权限 |
| Skill 安装 | 进度实时展示 |
| Skill 列表 | 卸载操作 |

### 2.5 Dashboard

| 功能 | 描述 |
|------|------|
| 任务统计 | 状态分布饼图 |
| 费用趋势 | 折线图 |
| 最近任务 | 快捷入口 |

### 2.6 数据导出

| 功能 | 描述 |
|------|------|
| 发起导出 | 配置时间范围、脱敏级别 |
| 进度查询 | 轮询状态 |
| 文件下载 | JSONL 格式 |

## 3. UI/UX 原则

- **开发者工具定位**：专业、克制、信息密度适中
- **配色**：Element Plus 默认主题 + 自定义 CSS 变量
- **响应式**：桌面优先，平板/移动端适配
